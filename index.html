Search Pointers

  • Type the first or last name, 2 letter state, district (ex. ID-2), committee name, or 5 digit zip code
  • Type a state's abbreviation to see all members from that state.
  • Type a state's abbreviation plus - to see all Congressional districts from that state (ex. 'KS-')
  • The border color changes with the Member's party affiliation.
  • The zip code search returns the House member. Note that a zip code search may not be 100% accurate.

This site is designed to give you quick access to key information on members of Congress and Committees. Right now it is focused on contact information, photos, and the commmittees they serve on. It is powered by data from the Sunlight Foundation and theunitedstates.io.

Information on Members of Congress is current. Committee membership is currently being updated.
